# PalindromeCheckerApp

A Java application to check if a given string is a palindrome.

## Project Structure
- **App**: Main application module containing the PalindromeCheckerApp class

## Author
Roll No: RA2411030010116

## Description
This application checks whether a given string reads the same forwards and backwards (palindrome).

## How to Run
1. Navigate to the App/src directory
2. Compile: `javac PalindromeCheckerApp.java`
3. Run: `java PalindromeCheckerApp`

## Use Cases
- **UC1: Application Entry & Welcome Message** - Displays a welcome message when the application starts
- **UC2: Hard Code Palindrome Check** - Tests predefined strings to check if they are palindromes
- **UC3: User Input Palindrome Check** - Allows users to input their own strings and check if they are palindromes

## Features
- Case-insensitive palindrome checking
- Ignores spaces in the input string
- Interactive user interface for multiple checks
- Comprehensive test cases with hard-coded examples

## Git Workflow
This project follows a feature-branch workflow:
